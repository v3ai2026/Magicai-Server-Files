# Server Directory

This directory contains the core server-side application logic for MagicAI.

## Structure

- **api/** - API endpoint handlers and logic
- **models/** - Database models and schemas
- **controllers/** - Business logic controllers
- **middleware/** - Express/server middleware functions
- **utils/** - Utility functions and helper modules
- **routes/** - Route definitions and endpoint mapping

## Purpose

This is the main backend codebase containing:
- API route handlers
- Database interaction logic
- Business logic implementation
- Authentication and authorization
- Error handling and validation
